# Запуск

Для запуска приложения, нужно запустить:

``` 
docker-compose.yml
```

Swagger доступен по адресу:

```
http://localhost:8095/swagger-ui/index.html#
```

Файлы для теста находятся по пути:

```
src/files
```

Пример входных данных:

```json
{
  "filePath": "numbers.xlsx",
  "number": 2
}
```
